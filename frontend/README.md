# Frontend (Vue 3)

SPA สำหรับทำข้อสอบ — รายละเอียดสถาปัตยกรรมและ flow อยู่ที่ **[`../docs/`](../docs/)** (ไม่ซ้ำในไฟล์นี้)

## Prerequisites

- [Node.js](https://nodejs.org/) LTS

## Commands

```bash
npm install
npm run dev          # http://localhost:5173
npm run build
npm run preview
```

## Layout

`src/views/` · `src/stores/` · `src/router/` · `src/assets/`
